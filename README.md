# FinQuest Financial Literacy Gamification Platform - Capstone Project

### Overview:

FinQuest is a full-stack gamified platform developed by Computer Science students at the University of Technology, Jamaica, to improve financial literacy through interactive modules like quizzes, crosswords and hangman. Built with a **React.js** frontend, a **Flask (Python)** REST API and a **MySQL** database, the system simulates real-world financial environments—including stocks, bonds and compound interest investments—while rewarding progress through a **Coin Credit System** and cosmetic store. The platform ensures data integrity and security for its users by utilizing **JWT authentication** and **bcrypt password hashing** to create a safe, engaging space for mastering personal finance.

### Features:

+ Interactive Learning Modules – Engage with financial concepts through a Quiz, Crossword and Hangman.

+ Simulated Fixed Deposits – Invest coins and calculate growth 

+ Stock Marketplace – Buy and sell shares while analyzing real-world metrics like ROE and Debt-to-Equity.

+ Coin Credit System – Earn digital currency through gameplay to fund investments or purchases.

+ Cosmetic Store – Spend earned coins to unlock and equip custom UI background colors.

+ Progress Tracking – Monitor XP, levels and quiz accuracy via a real-time user dashboard.

+ Leaderboards – View rankings based on total points and financial growth.

+ Admin CMS – Manage the system by adding, editing or deleting quiz questions and vocabulary.

+ User Authentication – Secure session management using JWT and bcrypt password hashing.


### Tech Stack:

+ React.js (Frontend) – Build a dynamic, responsive user interface for interactive game modules and real-time dashboards.

+ Flask (Backend) – Python-based REST API to handle mathematical simulations, investment logic and server-side processing.

+ MySQL (Database) – Relational data storage to maintain user profiles, transaction histories and game content with high integrity.

+ JWT (Authentication) – JSON Web Tokens for secure, stateless session management and role-based access control.

+ Bcrypt (Security) – Industry-standard password hashing to ensure user credential safety.

+ Axios (API Communication) – Facilitate seamless asynchronous data exchange between the React frontend and Flask backend.

+ SQLAlchemy (ORM) – Manage database interactions using Python objects for cleaner, more maintainable backend code.
